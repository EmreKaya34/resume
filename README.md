# Resume Website - GitHub Pages

This project is a simple resume/CV website prepared with HTML and CSS.

## Files

- `index.html`: Main resume page
- `style.css`: Design and responsive layout

## What to edit

Open `index.html` and change:

- Name
- About section
- Skills
- Projects
- Education
- Experience
- Email
- GitHub username
- Location

## GitHub Pages

Upload these files to a public GitHub repository, then enable GitHub Pages from:

Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: main → Folder: /root → Save

Your site URL will look like:

https://your-username.github.io/repository-name/

If your repository name is `your-username.github.io`, your URL will be:

https://your-username.github.io/
